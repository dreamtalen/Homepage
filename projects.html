Image Super-Resolution Using Convolutional Neural Network(CNN)

Supervised by Prof. Haibao Chen, Shanghai Jiao Tong University

  • Applied deep learning techniques to sharpen or improve the quality of a low-resolution image input by outputting a super-resolved high-resolution image.
  • Proposed three different CNN models and the 7-layers CNN_DEEP got lower loss value and reduced 10% training time, compared with "waifu2x", a open source image super-resolution software.



Three-dimensional Integrated Circuit(3D-IC) Partitioning Technique Design

Supervised by Prof. Weifeng He, Shanghai Jiao Tong University

  • Excellent graduate thesis in Department of Micro-Nano Electronics, Shanghai Jiao Tong University.
  • Designed a 3D-IC partitioning algorithm for motion estimation(ME) module of HEVC, including the initial partition algorithm based on breath first search and the iterative optimization algorithm.
  • Designed and implemented a partitioning algorithm evaluation platform which that has functions including a netlist file parser, analyzing area reports, modeling circuits with hyper-graph data structures, applying partitioning algorithms, and evaluating partition results by the area utilization rate and cut edges number.
  • Achieved an optimum partition for the ME module, which reduced 42.9% of the cut edges while the area utilization rate reached 95.5%.
  • Code repository can be founded here at Github.com.
  • This paper can be downloaded here (Chinese version).
